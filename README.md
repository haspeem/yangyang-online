# yangyang-online

基于 [a835100635/yangyang](https://github.com/a835100635/yangyang) 改造的移动端可连接版。

原项目是一个简易的"羊了个羊" HTML 游戏，感谢原作者的开源分享 🙏

## 本项目改动

- 适配移动端布局
- 改造为移动端 APP 可连接版本
- 后续计划加入网络联机功能

## 原项目功能

- 随机卡牌布局 + 层级遮挡
- 点击卡牌移入卡槽，三张相同消除
- 关卡递进，图标随关卡增加
- 背景音乐

## 本地运行

直接用浏览器打开 `index.html` 即可。
